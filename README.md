# Lógica de Programação - IFSP (SPOLOG1)

Repositório destinado ao armazenamento de exemplos práticos e exercícios desenvolvidos em Linguagem C durante o primeiro semestre.

## 🛠️ Tecnologias
- **Linguagem:** C
- **Editor de Código:** Visual Studio Code

## 📁 Estrutura do Repositório
* **`aula01_introducao`**: Primeiros algoritmos e lógica básica.
* **`aula02_fluxogramas`**: Implementações iniciais com desvios condicionais (`if`).
* **`aula03_condicionais_repeticao`**: Prática com `switch..case` e laços `while`.
* **`aula04_do_while`**: Exercícios com estrutura de repetição `do..while`.
* **`aula05_for_vetores`**: Manipulação de arrays e laços `for`.
